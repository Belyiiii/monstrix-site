// MONSTRIX — варианты wordmark с O = глаз

Идея: O в названии — глаз монстрика (как на твоих ESP32-устройствах). Ниже 8 концепций. Каждая показывает живую анимацию (моргание, движение зрачка, глитч). К каждой — ASCII-вариант для Telegram/чатов и описание как порт под ESP32 (bitmap PROGMEM, U8g2 / Adafruit_GFX, или Unicode).

eye_o_inline
v1 // css
MNSTRIX

Концепт: чистый wordmark, буква O — настоящий круг с зрачком, зрачок бегает по сторонам, веко моргает раз в 5 сек.
Сильная сторона: работает в любом размере, идеален для шапки сайта и favicon.
ESP32: рисуется через `tft.drawCircle()` + `tft.fillCircle()` для зрачка. На OLED 128×64 — 12px кружок с 4px зрачком.

<div class="logo-eye">M<span class="eye"></span>NSTRIX</div>

/* ESP32 (TFT_eSPI) */
tft.drawCircle(cx, cy, R, GREEN);     // контур O
tft.fillCircle(cx + dx, cy, r, GREEN); // зрачок (dx = -R/2..R/2)

Концепт: текущий блочный лого, но в нижней половине O — зрачок-точка ●, который моргает (исчезает) и иногда краснеет (cyan). Чистый текст + один эмулированный пиксель.
Сильная сторона: не ломает существующий лого, идеален для terminal-темы Hugo, копируется как текст.
ESP32: печатается шрифтом, точка — отдельный `drawPixel()` который мигает по таймеру.

icon_plus_word
v3 // mascot
MONSTRIX

Концепт: глаз вынесен слева как mascot-иконка, рядом wordmark. Это твой исходный образ — «монстрик с глазом». Глаз можно использовать отдельно (favicon, иконка приложения, sticker для Telegram).
Сильная сторона: отделимый mascot. Глаз становится главным брендовым символом, его легко рисовать на корпусах устройств и на 3D-печати.
ESP32: идеально под главную «idle»-анимацию — глаз озирается, моргает, реагирует на тач/звук. 

// 3 строки — для большинства OLED 128x64

tri_eye_monogram
v4 // favicon

Концепт: три глаза в ряд — как 3 устройства из линейки MONSTRIX (NeuroFrame, Cardputer, etc). Моргают по-очереди. Идеально под app-icon/favicon — узнаваемая монограмма.
Сильная сторона: работает в маленьком размере (32×32), не зависит от шрифтов, легко рисуется в Affinity/Figma.
ESP32: «splash-screen» при загрузке — три глаза моргают по-очереди, потом гаснут.

U+25EFM◯·NSTRIXcircle + middle-dot

Концепт: один Unicode-глиф как глаз — самый минималистичный и портативный вариант. Копируется в любой чат/тайтл вкладки/имя бота. Фаворит — M◉NSTRIX (U+25C9 FISHEYE) — выглядит как закрашенный глаз и читается как «O».
ESP32: работает только если в шрифте есть этот глиф (U8g2 имеет несколько шрифтов с symbol-блоками). Фоллбек — нарисовать `drawCircle + fillCircle`. 

// Текстовые формы для копирования:
M◉NSTRIX     // fisheye, идеален
M⊙NSTRIX     // circled dot
M⊚NSTRIX     // circled ring
MΘNSTRIX     // theta — «закрытый глаз»
big_eye_centerpiece
v6 // poster
MONSTRIX

Концепт: крупный глаз сверху, wordmark под ним. Работает как splash-screen на ESP32 при загрузке устройства, как hero блока сайта и как обложка соцсетей.
Сильная сторона: вертикальная композиция занимает мало ширины, легко вписывается в квадратные дисплеи 240×240 (GC9A01, ST7789).
ESP32: 1 эллипс + 1 закрашенный круг + текст. На большом экране — главный «лицевой» лого.

slash_eye
v7 // tag
MONSTRIX //

Концепт: wordmark — обычный текст, после `//` идёт тэг-глаз янтарного цвета. Семантика: «MONSTRIX // [eye]» = бренд и его watcher-mascot. Хорошо для футеров, email-подписей, commit-message в репозитории.
ESP32: в режиме «info bar» внизу экрана. 

MONSTRIX // (●)        // ASCII для чата
MONSTRIX // ◉
M0NSTRIX // [●]
pixel_eye_8x8
v8 // esp32

OPEN

HALF

MONSTRIX

Концепт: попиксельный глаз 8×8 — точно так же выглядит на OLED SSD1306 (128×64). Три кадра моргания → готовая sprite-анимация.
ESP32 PROGMEM код рядом — копируй в `eye.h` и рисуй через `u8g2.drawXBMP(x, y, 8, 8, eye_open)`. 

// eye.h — 8x8 monstrix eye sprites
const uint8_t eye_open[] PROGMEM = {
  0b00111100,
  0b01000010,
  0b10011001,
  0b10111101,
  0b10111101,
  0b10011001,
  0b01000010,
  0b00111100,
};
const uint8_t eye_half[] PROGMEM = {
  0b00000000,
  0b00111100,
  0b01111110,
  0b11111111,
  0b11111111,
  0b01111110,
  0b00111100,
  0b00000000,
};
const uint8_t eye_closed[] PROGMEM = {
  0b00000000,
  0b00000000,
  0b00000000,
  0b00111100,
  0b01111110,
  0b00111100,
  0b00000000,
  0b00000000,
};

// usage (U8g2):
void drawLogo() {
  u8g2.drawXBMP(0, 0, 8, 8, eye_open);
  u8g2.setFont(u8g2_font_6x10_mr);
  u8g2.drawStr(12, 8, "MONSTRIX");
}
