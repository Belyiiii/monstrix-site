## Diymore 5V 2.4A Type-C — лучший выбор

![Diymore 5V 2.4A](/images/components/diymore-5v-24a.webp)

**Рекомендую!** Всё в одном модуле — зарядка, защита, boost до 5V.

**Характеристики:**

- Вход: Type-C, 5V
- Выход: 5V / 2.4A (стабильные!)
- Зарядка: до 2.4A
- КПД: до 96%
- Защита: перезаряд, переразряд, КЗ
- Индикация: 4 светодиода (уровень заряда)
- Размер: 17.5×10×6 мм

**Почему он лучший:**

- Один модуль решает всё
- Стабильные 5V на выходе → подаёшь на VIN ESP32
- Компактный
- Type-C — современно

**Подключение:**

```
Li-Ion 18650
├── (+) → + модуля
└── (-) → - модуля

Модуль 5V → VIN ESP32
Модуль GND → GND ESP32
```

## TP4056 + Boost 5V 1A — бюджетный вариант

![TP4056 Boost](/images/components/tp4056-boost.webp)

Комбо-модуль: зарядка TP4056 + повышайка до 5V в одной плате.

**Характеристики:**

- Вход: Type-C, 5V
- Выход: 5V / 1A
- Зарядка: TP4056
- Защита: перезаряд, КЗ
- Размер: ~20×20 мм

**Плюсы:**

- Дешевле чем Diymore
- Компактный
- Всё в одном

**Минусы:**

- Ток только 1A (для ESP32 хватит, но без запаса)
- Качество зарядки среднее

**Для кого:**

Простые проекты — ESP32 + пара датчиков. Если нужен TFT дисплей или много светодиодов — бери Diymore.

**Ссылки:**

- [AliExpress](https://ali.click/tbijz7?erid=2SDnjeJhcWV)

## MT3608 DC-DC Boost — если уже есть TP4056

![MT3608](/images/components/mt3608.webp)

Повышающий преобразователь. Превращает плавающие 3.0-4.2V от TP4056 в стабильные 5V.

**Характеристики:**

- Вход: 2V - 24V
- Выход: 5V - 28V (настраивается потенциометром)
- Ток: до 2A
- КПД: до 96%
- Размер: 37×17×14 мм

**Как настроить:**

1. Подключи вход к источнику (или TP4056)
2. **Не подключай ESP32!**
3. Крути потенциометр **против часовой** (~20 оборотов)
4. Измерь выход мультиметром
5. Добейся ровно 5.0V
6. Теперь можно подключать ESP32

⚠️ **Важно:** Настрой напряжение ДО подключения к ESP32! Можно случайно подать 12V и спалить плату.

**Схема с TP4056:**

```
18650 → TP4056 → MT3608 → ESP32
         B+/B-    IN+/IN-   VIN
                  OUT+/OUT-
                    ↓
               Настрой 5V!
```

**Для кого:**

У тебя уже есть TP4056 и не хочешь покупать новый модуль — добавь MT3608 и получишь стабильные 5V.

## Что НЕ подходит для ESP32

### TP4056 без boost

Выдаёт плавающее напряжение 3.0-4.2V — ESP32 будет работать нестабильно.

### Модули только для зарядки

Если в описании нет слов "boost", "step-up", "5V output" — скорее всего на выходе плавающее напряжение.

## Схемы подключения

### Вариант 1: Diymore / TP4056+Boost (рекомендую)

```
Li-Ion 18650
├── (+) → B+ модуля
└── (-) → B- модуля

Модуль:
├── 5V OUT → VIN ESP32
└── GND    → GND ESP32

Type-C → зарядка
```

### Вариант 2: TP4056 + MT3608

```
Li-Ion 18650
├── (+) → B+ TP4056
└── (-) → B- TP4056

TP4056:
├── OUT+ → VIN+ MT3608
└── OUT- → VIN- MT3608

MT3608 (настроен на 5V):
├── VOUT+ → VIN ESP32
└── VOUT- → GND ESP32
```
