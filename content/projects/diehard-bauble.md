Классический ёлочный шар превращён в миниатюрный видеоплеер. Внутри — ESP32-S3 и дисплей ST7789 1.9", который крутит культовую сцену с зажигалкой из «Крепкого орешка». Видео любое: до 40 секунд при 13 FPS.

## [#] Характеристики

| Параметр | Значение |
|---|---|
| Дисплей | ST7789 1.9" TFT, 320×170 |
| Контроллер | ESP32-S3 (4 МБ Flash, 2 МБ PSRAM) |
| Видео | до 40 сек, 13 FPS |
| Питание | Li-Ion, зарядка USB-C через TP4056 |
| Автономность | ~2–3 часа |
| Вес | ~80 г с батареей |

## [+-] Подключение

```
ESP32-S3 → ST7789
─────────────────
3V3 → VCC      10 → CS
GND → GND       9 → DC
12  → SCL       8 → RST
11  → SDA      13 → BLK

Питание:  Battery+ → [тумблер] → B+ ,  Battery- → GND
```

[!] Тумблер ставится на плюсовой провод.
